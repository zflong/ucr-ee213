Under Linux environment, use command:

make parse
make runparse

to compaile the codes;

Use command:

./runparse.exe filename.sp

To execute the codes.

As will show in the screen:
First part is netlist contains device table, node table and amount of each type of device
Second part is node name table and node index table
Third part is information about matrixsize and node number 
Last part is MNA matrix under S domain.
